# Интернет-магазин одежды

Веб-платформа для онлайн-торговли, разработанная в рамках учебной практики.

## Технологии

- Python, Django
- MySQL
- HTML, CSS, JavaScript

## Функциональность

- Каталог товаров с фильтрацией по категориям, сортировкой и пагинацией
- Карточки товаров с подробной информацией
- Корзина: добавление, изменение количества, удаление товаров
- Оформление заказа на основе корзины
- Регистрация и авторизация пользователей
- Административная панель Django для управления товарами и категориями

## Архитектура

- Клиент-серверная модель
- Компоненты: веб-сервер, Django-приложение, база данных
- Модель данных: таблицы для товаров, категорий, пользователей, заказов

## Тестирование

Написаны и выполнены функциональные тесты JUnit для проверки бизнес-логики
